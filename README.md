# Laboratory Work #4

**Student Variant:** 11

## Description
This repository contains solutions for Laboratory Work #4 on Java Arrays.

### Tasks Implemented:
1.  **Task 11:** Find the sum of the largest and smallest elements of the array.
    * *Class:* `Task11.java`
    * *Implementation:* Uses a `for-each` loop.
2.  **Task 41:** Determine the arithmetic mean of the maximum and third elements of the array.
    * *Class:* `Task41.java`
    * *Implementation:* Uses a standard `for` loop. Checks if array length >= 3.
3.  **Task 64:** Create a new array based on rules (negative -> positive, positive -> decrease by 3, zero -> -2).
    * *Class:* `Task64.java`

## How to Run
Each class contains a `main` method with predefined test cases. You can run them individually in any Java IDE or via command line:

javac src/main/java/lab4/*.java
